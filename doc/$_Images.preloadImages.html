(static) preloadImages(images, callbackopt, nullable, returnCollectionopt, nullable) → {Promise|Object.<String, String>|Image}

Preloads images by URL. Images can be preloaded by name and are thereby retrievable afterwards or anonymously. So you can either just use the url again, or, to be super-sure, call the method again, with just the image name to get the URL.
Parameters:
Name Type Attributes Default Description
images String | Array.<String> | Object.<String, String> an URL, an array of URLS or a plain object containing named URLs. In case the string is an already used name, the image-object is returned.
callback function <optional>
<nullable>
 callback to call when all images have loaded, this also fires on already loaded images if inserted again
returnCollection Boolean <optional>
<nullable>
 false defines if the function should return the collection in which the images where inserted instead of the promise object

Returns:
either returns a promise object (does not fail atm), the currently added named/unnamed images as saved (if defined by returnCollection) or a requested cached image
Type
Promise | Object.<String, String> | Image
