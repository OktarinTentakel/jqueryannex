(static) contextIsTouchDevice(inspectUserAgentopt, nullable, additionalUserAgentIdsopt, nullable, onlyConsiderUserAgentopt, nullable) → {Boolean}

Detects if the current JavaScript-context runs on a (dedicated) touch device. Checks these UserAgents by default: iOS-devices, Blackberry, Android, IE mobile, Opera Mobilem Firefox Mobile and Kindle.
Parameters:
Name Type Attributes Default Description
inspectUserAgent Boolean <optional>
<nullable>
 false defines if the user agent should be inspected additionally to identifying touch events
additionalUserAgentIds Array.<String> <optional>
<nullable>
 list of string-ids to search for in the user agent additionally to the basic ones
onlyConsiderUserAgent Boolean <optional>
<nullable>
 false tells the algorithm to ignore feature checks and just go by the user-agent-ids
