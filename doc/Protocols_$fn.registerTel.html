Methods

(static) registerTel(regionPart, firstTelPart, countryPart, secondTelPart, writeToElemopt, nullable, eventTypeopt, nullable) → {Object}

Register an event handler to activate a tel-protocol phonecall without openly writing down the number. Parameters mixed to complicate parsing.
Parameters:
Name Type Attributes Default Description
regionPart Number.Integer | String the local part of the number after the country part e.g. +49(04)<-this 123 456
firstTelPart Number.Integer | String first half of the main number +4904 (123)<-this 456
countryPart Number.Integer | String the country identifactor with or without + this->(+49)04 123 456
secondTelPart Number.Integer | String second half of the main number +4904 123 (456)<-this
writeToElem Boolean <optional>
<nullable>
 false define if the number should be written back to the element text
eventType String <optional>
<nullable>
 click the event type to register the call to
