(static) waitForWebfonts(fonts, callbacknullable, fallbackFontNameopt, nullable) → {Promise}

Waits for a list of webfonts to load before executing a callback. Works for fonts already loaded as well.
Parameters:
Name Type Attributes Default Description
fonts String | Array.<String> the CSS-names of the fonts to wait upon
callback function <nullable>
 the callback to execute once all given webfonts are loaded
fallbackFontName String <optional>
<nullable>
 sans-serif the system font onto which the page falls back if the webfont is not loaded
